Projeto Belém Digital Web

BELÉM DIGITAL WEB

Belém Digital Web (BDW) é um projeto inovador que oportuniza aos microempreendedores e empresas uma abordagem a mais de adquirir clientes na internet de forma estratégica.

Serviços

Landing Page (Página de destino)

Ela é uma página única na web com o objetivo de conversar com o visitante de forma direta a fim de torná-lo em um cliente digital.

1. Atende às necessidades específicas do seu cliente;
2. Melhora os resultados da sua campanha de publicidade;
3. Deixa uma boa primeira impressão;
4. Constrói sua credibilidade.

Aplicativo (APP)

Um APP promove produtos e serviços de forma ágil, oferecendo conexão com o público e possibilita qualidade de trabalho interno.

1. Fideliza e relaciona-se com o cliente.
2. Notifica promoções, eventos e lançamentos.
3. Facilita a obtenção de dados e automatização de processos.
4. Oportuniza a utilização de um canal de sugestões.

As linguagens de programação que domina são Python e JavaScript. Além disso, ele tem conhecimento sobre TypeScript, React e Node.js.
